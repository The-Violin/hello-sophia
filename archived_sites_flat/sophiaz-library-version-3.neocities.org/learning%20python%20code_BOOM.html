Step 4: Replace Multiple Words at Once (God, Jesus, Spirit)

You will eventually want to replace many theological terms. Here is a script that does that:



python

# Replace multiple words in one pass

replacements = {

    "God": "God: DDC-231",

    "Jesus": "Jesus: DDC-232",

    "Christ": "Christ: DDC-232",

    "Holy Spirit": "Holy Spirit: DDC-231",

}



text = "God sent Jesus Christ. The Holy Spirit is with us."



for old, new in replacements.items():

    text = text.replace(old, new)



print(text)

Output:



text

God: DDC-231 sent Jesus: DDC-232 Christ: DDC-232. The Holy Spirit: DDC-231 is with us.

ð Step 5: Read from a File and Save to a New File (The Real Tool)

This is the version you will use for your library pages. It reads a .txt file, replaces the words, and saves a new file.



python

# File-based replacement script

import re



# Define replacements (add as many as you like)

replacements = {

    r'(?i)\bGod\b': 'God: DDC-231',

    r'(?i)\bJesus\b': 'Jesus: DDC-232',

    r'(?i)\bChrist\b': 'Christ: DDC-232',

}



# Ask for the file name

input_file = input("Enter the name of your text file (with .txt): ")



try:

    # Read the original file

    with open(input_file, 'r', encoding='utf-8') as file:

        content = file.read()

    

    # Apply each replacement

    for pattern, replacement in replacements.items():

        content = re.sub(pattern, replacement, content)

    

    # Create a new file name

    output_file = input_file.replace('.txt', '_DDC231.txt')

    

    # Save the new content

    with open(output_file, 'w', encoding='utf-8') as new_file:

        new_file.write(content)

    

    print(f"Success! New file saved as: {output_file}")

    

except FileNotFoundError:

    print("File not found. Please check the name and try again.")
